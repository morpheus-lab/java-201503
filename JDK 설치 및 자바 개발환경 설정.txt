Java 개발 환경을 설정하는 방법
[JDK 설치]
1. 웹브라우저로 java.oracle.com 으로 이동
2. Top Downloads > Java SE 링크 클릭
3. JDK 아래 "Download" 버튼 클릭
4. 라이선스 동의 클릭
5. 사용하는 OS에 맞는 설치파일 다운로드 후 실행

[환경변수설정]
1. 탐색기 등에서 "내 컴퓨터" 오른 클릭 후 속성 선택
2. 시스템 정보 화면 왼쪽에 있는 "고급 시스템 설정"을 클릭하면 "시스템 속성"창 등장
3. "고급" 탭 아랫부분에 있는 "환경 변수" 버튼을 클릭
4. "환경 변수" 설정 창의 하단에 있는 시스템 변수에 "새로 만들기" 클릭
5. 변수 이름: "JAVA_HOME"
   변수 값: 자바 설치 경로
6. "확인" 클릭
7. "Path"라는 시스템 변수를 찾아 더블 클릭
8. 변수 값에 있는 기존 값을 지우지 말고, 커서를 맨 이동시킴
9. "%JAVA_HOME%\bin;{기존 값}" 입력 후 확인
